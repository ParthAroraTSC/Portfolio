# NOTREGRETO Portfolio

A professional, high-fidelity Cybersecurity Portfolio designed with a custom Ubuntu 20.04 (Linux) theme. This project showcases technical expertise in SOC Analysis, VAPT, and Network Security through an interactive desktop environment.

## 🚀 Overview

Developed and maintained by **Parth Arora**, this portfolio provides a unique way to explore professional experience, technical projects, and security reports. Built with **Next.js**, **Tailwind CSS**, and **React**, it offers a fully functional desktop experience with window management, a terminal emulator, and custom application modules.

## 🛠️ Key Features

- **Interactive Desktop**: A complete Ubuntu-inspired UI with window management and system utilities.
- **Cybersecurity Projects**: Dedicated showcase for security-focused implementations and technical reports.
- **Terminal Emulator**: Functional terminal allowing users to explore the filesystem and run custom commands.
- **Brand Identity**: Personalized "PA" (Parth Arora) branding integrated across the OS favicon and system assets.
- **Modular Data Layer**: Clean separation of content and UI for easy maintenance and scalability.

## 💻 Tech Stack

- **Framework**: Next.js (React)
- **Styling**: Tailwind CSS & Vanilla CSS
- **Interactions**: Framer Motion & GSAP
- **State Management**: React Hooks & Local Storage
- **Analytics**: Google Analytics 4

## 📂 Project Structure

- `/components`: Modular UI components and system applications.
- `/data`: Centralized data modules for projects and reports.
- `/public`: Static assets, brand logos, and technical files.
- `/styles`: Global CSS and theme-specific styling.
